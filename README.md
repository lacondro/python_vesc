
# Project Title

Modified version of https://github.com/LiamBindle/PyVESC.

Have more features:

-get mcconf

-set mcconf






## Install with pip

install this custom version of pyvesc to freely communicate with VESC

```bash
  pip install git+https://github.com/lacondro/python_vesc.git
```
